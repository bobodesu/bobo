## 判断奇数偶数
```cpp
bool isEven(int num){
    return !(num&1);
}
```

## 交换变量
```cpp
void swap(int &a,int &b){
    a^=b;
    b^=a;
    a^=b;
}
```

## 正负号互换
```cpp
int signReversal(int num){
    return ~num+1;
}
```

## 大小写转换
利用位运算，以高效的运算速度实现大小写转换  
```cpp
char toLower(char ch){
    return ch|32;
}
char toBigger(char ch){
    return ch&223;
}
```

## 字符数组转整数
```cpp
int stringToInt(char str[],int length){
    int result=0;
    while(length--){
        result*=10;
        result+=(str[length]&207);
    }
    return result;
}
```
